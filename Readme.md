## 🌾 Agro Web Application

A web-based platform developed to help farmers access crop information and government schemes easily.
This application allows Admins to manage crops and schemes, and Farmers (Users) to view details and apply for schemes online.

The main goal of this project is to simplify agricultural information management and provide a digital solution for farmer-government interaction.



## 📂 Repository Link: 
https://github.com/Chaitanya753/Agro_Web_App.git

👨‍💼 Admin Panel

## 📦 Installation & Setup
1️⃣ Install XAMPP

Download and install XAMPP.

2️⃣ Clone Repository
git clone = https://github.com/Chaitanya753/Agro_Web_App.git

3️⃣ Move Project Folder

Place the project inside:

C:\xampp\htdocs\
4️⃣ Create Database

Open phpMyAdmin

Create database: agro_web_app

Import the provided SQL file

5️⃣ Run Project

Start Apache and MySQL in XAMPP.

Open browser:

http://localhost/Agro_Web_App/
🗄️ Database Structure
Tables:

users

admin

crops

schemes

applications

Relationships:

Users apply for Schemes

Admin approves/rejects applications
